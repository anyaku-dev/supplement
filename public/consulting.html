Value Proposition

何が手に入るか

01

多忙な日々でも自宅・オンラインでコンディションを可視化

腸内フローラ、血糖値、ホルモン、栄養素と４つの軸で身体の状態を可視化。専門家のフィードバックにより、何をすべきかが具体的に分かります。

半年ごとの検査・レビューでBEFORE→AFTERまで伴走

主観ではなく、科学的根拠に基づいたフィードバック。コンディションの最適化により、仕事のパフォーマンスを安定させます。

検査・分析ツール

腸内フローラ検査

年2回 (半年ごと)

身体の土台となる腸内環境を可視化

  • ・自宅で採便→ポスト投函
  • ・パフォーマンススコア（糖代謝 / ダイエット / 疲労耐性 / 睡眠の質 / 持久力）
  • ・善玉菌 / 悪玉菌の有無と解説
  • ・腸内で生み出す各種栄養成分（短鎖脂肪酸 / 乳糖 / ポリアミン / GABA / ビタミンB群）

ホルモン検査

年2回 (半年ごと)

身体の機能調整力を把握

  • ・唾液を専用容器へ採取→ポスト投函
  • ・メラトニン（睡眠の質・メンタルヘルスに寄与）
  • ・コルチゾール（炎症抑制、免疫調整、主にストレス反応に関与）
  • ・テストステロン（気力、集中力、体作りに寄与）

血糖値検査キット

1回10日間/年2回 (半年ごと)

エネルギーの波とコンディションを把握

  • ・センサー機器を装着
  • ・「点」の血糖値ではなく、10日間、5分毎の血糖値推移をモニタリング
  • ・数値の変動を自身の体調と重ねて特徴を把握（食事も同時に記録）

栄養素分析

1回10日間/年2回 (半年ごと)

日々の食生活での栄養バランスを可視化 (血糖値検査と同時に実施)

  • ・専用アプリを活用
  • ・カロリー / タンパク質 / 脂質 / 炭水化物 / 塩分 / 糖質 / 食物繊維を数値化
  • ・血糖値データとの相関とも連携

サプリメント

発酵高麗人参

毎月2箱60包お届け

日々の習慣の軸となるサプリメント

  • ・豊富なエビデンスを有する高麗人参に着目
  • ・有効成分「コンパウンドK」の配合量を最重視（1包15mg / 朝夕1日2包）
  • ・発酵による吸収率の大幅改善（非発酵に比べ118.3倍）
  • ・健康の鍵である腸内環境にも配慮（甘味料Free）

※ 検査は診断ではなく、健康状態の参考情報としてご提供します
